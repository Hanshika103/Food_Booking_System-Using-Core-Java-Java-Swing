# Campus Food Booking System

## 💡 Problem Statement

During college exam periods, students often face difficulty in ordering food as they are busy with studies and cannot leave their rooms or classrooms to book food manually. Traditional methods of ordering food are time-consuming, inconvenient, and lack proper organization.

This creates a need for a digital solution that allows students to easily browse food items, place orders, and manage their requests without disrupting their study schedule.

---

## 🎯 Project Objective

To develop a food booking system that enables students to order food online quickly and efficiently during busy academic schedules, especially exams, reducing manual effort and saving time.

---

## 🚀 Project Description

The Food Booking System is designed to simplify the process of ordering food for college students. It provides a user-friendly interface where users can view available food items, add them to a cart, and place orders seamlessly.

The system also includes an admin module to manage food items and ensures that all data such as user details and orders are stored and managed efficiently using a database. This project focuses on improving convenience, reducing time consumption, and enhancing user experience in a real-world college environment.


## 🚀 Features

* 👤 **User Authentication**
  Secure login and registration system for users.

* 🍽 **Food Menu Display**
  Displays available food items with details such as name, price, and category.

* 📦 **Direct Order Placement**
  Users can directly place an order for selected food items without using a cart system.

* 🔄 **Order Management**
  Stores and manages order details efficiently in the database.

* 🛠 **Admin Controls**
  Admin can add, update, and delete food items from the system.

* 🗄 **Database Integration**
  Uses MySQL to store user data, menu items, and order information.

* 🖥 **User-Friendly Interface**
  Simple and interactive interface built using Java Swing.
